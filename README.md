# Arham Ansari - GeoAI Portfolio

Professional portfolio for Arham Ansari, GeoAI & Satellite ML Engineer.

## Technical Stack
- **Frontend**: React, Tailwind CSS, Lucide Icons
- **Deployment**: Cloudflare Pages
- **Build Tool**: Vite

## Project Focus
- Satellite ML Pipelines (NASA IMERG, SMAP)
- Flood Risk Forecasting (GeoRiskAI)
- Urban Heat Island Analysis

## Local Development
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
